Gem Data Cleaning Project
Description: This script automates the process of cleaning raw Gem data from CSV files.

What it does:

Reads semicolon-separated (;) CSV files.

Cleans column names (strips whitespaces).

Selects key features (GemName, Usage, Downloads).

Rounds numerical values to 4 decimal places.

Handles missing values (removes empty rows).

How to run:

Install pandas: pip install pandas

Run script: python data_cleaner.py
